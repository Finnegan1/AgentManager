title: Managing Skills

The **Skills** page lets you manage your skill library — create new skills, install from the marketplace, edit existing ones, search through them, and delete those you no longer need.

## Skill List

Skills are displayed as cards in a responsive grid layout (1–3 columns depending on screen width). Each card shows:

- **Name** and **description** (clamped to 2 lines)
- **Tags** as colored badges
- A **Claude Code** badge if the skill uses Claude Code-specific fields
- **Version** and **author** at the bottom
- **Actions** via dropdown menu: View, Edit, Delete

Use the **search box** to filter skills by name, description, or tags.

## Installing from the Marketplace

You can install skills from [skills.sh](https://skills.sh/), Anthropic's public skill registry:

1. Click **Skill Marketplace** to open skills.sh in your browser
2. Browse or search for a skill
3. Copy the install command shown on the skill page (e.g. `npx skills add https://github.com/shadcn/ui --skill shadcn`)
4. Paste the command into the install input field in the app
5. Click **Installieren** (Install)

The app fetches the `SKILL.md` from the GitHub repository, saves it to your skills directory, and syncs symlinks automatically. The skill appears in your list immediately.

<Callout>
  Skills from the marketplace use the same `SKILL.md` format. The install command is parsed to extract the GitHub repository URL and skill name.
</Callout>

## Creating a Skill

1. Click **Skill erstellen** (Create Skill)
2. A near-full-screen editor dialog opens
3. Enter the **Skill-ID** — this becomes the directory name (e.g. `typescript-best-practices` creates `typescript-best-practices/SKILL.md`)
4. Edit the content in the CodeMirror editor. The template includes YAML frontmatter:

```markdown
---
name: ""
tags: []
description: ""
version: "1.0"
author: ""
created: "2026-03-22"
updated: "2026-03-22"
---

# Skill Title

Beschreibe den Skill hier...
```

5. Click **Speichern** (Save)

The skill is saved to `~/.agent-manager/skills/{id}/SKILL.md` and immediately available through the MCP gateway and Claude Code (via symlink to `~/.claude/skills/`).

## Editing a Skill

Click the dropdown menu on a skill card and select **Bearbeiten** (Edit). The same editor dialog opens with the skill's current content. The Skill-ID is locked (not editable).

## The Editor

The skill editor uses **CodeMirror 6** with several features:

| Feature | Description |
|---|---|
| **Vim keybindings** | Full Vim mode — normal, insert, visual, and command-line modes |
| **Markdown highlighting** | Syntax highlighting for headings, bold, italic, links, code blocks |
| **YAML frontmatter** | Highlighted metadata section at the top of each skill |
| **Line numbers** | Gutter with line numbers and active line highlighting |
| **Code folding** | Collapse sections with the fold gutter |
| **Search** | Built-in search and replace (use `:` commands in Vim or Ctrl/Cmd+F) |
| **Bracket matching** | Auto-matching and auto-closing of brackets |
| **Line wrapping** | Long lines wrap to fit the editor width |

<Callout>
  The editor opens in **Vim normal mode** by default. Press `i` to enter insert mode, `Esc` to return to normal mode. Use `:w` to save (via the Save button) or standard Vim motions to navigate.
</Callout>

## Viewing a Skill

Click the dropdown menu and select **Anzeigen** (View) to see the skill's raw markdown content in a read-only dialog. This is useful for quick reference without entering the editor.

## Deleting a Skill

Click the dropdown menu and select **Löschen** (Delete). The skill file is removed from disk. This action cannot be undone.

## Search

The search box filters skills in real-time. It matches against:
- Skill **name**
- Skill **description**
- Skill **tags**

The search is case-insensitive and updates the displayed cards as you type.
