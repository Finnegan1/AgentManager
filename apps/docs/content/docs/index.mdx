**Skill Gateway** is a desktop application and MCP (Model Context Protocol) gateway that lets you manage reusable AI skills as markdown documents and proxy multiple downstream MCP servers through a single, unified endpoint. Instead of connecting your AI client to many servers individually, you connect it to Skill Gateway once — and it handles the rest.

## Key Features

- **Skill Management** — Create, edit, search, and organize skills as markdown files with YAML frontmatter metadata
- **MCP Gateway** — Proxy tools, resources, and prompts from multiple downstream MCP servers through one connection
- **Desktop App** — A native Tauri application with a full-featured dashboard for managing everything
- **Vim-Powered Editor** — Built-in CodeMirror 6 editor with Vim keybindings for editing skill content
- **Hot-Reload Config** — Change server configurations on the fly; the gateway picks up changes automatically
- **Three Transport Types** — Connect to downstream servers via stdio, SSE, or streamable HTTP
